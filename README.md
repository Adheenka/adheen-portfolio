# Mohammed Adheen — Portfolio Website

A modern, premium dark-themed portfolio website built with HTML, CSS, and Vanilla JavaScript.

## 🌐 Live Site
Once deployed: `https://Adheenka.github.io/adheen-portfolio`

## ✨ Features
- **Premium dark design** with purple/cyan gradient accents
- **Particle background** with animated connections
- **Custom cursor** with smooth follower
- **Role rotator** – cycles through job titles
- **Scroll reveal animations** for all sections
- **Skill progress bars** (animated on scroll)
- **Fully responsive** – mobile & desktop
- **Contact form** (opens email client via mailto)

## 📁 Project Structure
```
adheen-portfolio/

└── .github/
    └── workflows/
        └── deploy.yml  # GitHub Pages auto-deploy
```

## 🚀 Deployment (GitHub Pages)

### Setup (One-time)
1. Push code to `main` branch on GitHub
2. Go to **Settings → Pages** in your repository
3. Set **Source** to `GitHub Actions`
4. The workflow will auto-deploy on every push!

### Update Site
```bash
git add .
git commit -m "Update portfolio"
git push origin main
```
